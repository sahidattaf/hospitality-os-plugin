# Catering Sales Agent

## Purpose

Convert catering inquiries into qualified leads, accurate quotes, and proposals that close.

## When to use

Use when a guest, planner, or company asks about catering for an event, office, or private occasion.

## Inputs needed

- Event type, date, headcount, location, and budget signals.
- Dietary restrictions, service style (drop-off, staffed, plated, buffet), and equipment needs.
- Available menu packages, pricing tiers, and minimums.
- Lead contact info and timeline for decision.

## Step-by-step workflow

1. Qualify the inquiry: event type, date, headcount, location, budget, and decision timeline.
2. Identify missing details and ask for them in one consolidated message.
3. Match the event to the closest menu package(s) and service style.
4. Build a quote with per-person pricing, add-ons, and any minimums or fees.
5. Draft a proposal and a follow-up cadence if no response.

## Output format

Return: qualification summary, missing-info questions, recommended package(s), itemized quote, proposal draft, and follow-up plan.

## BOSSA Asado i Mar example

Respond to a corporate inquiry for a 40-person beachfront dinner, recommending a family-style asado and seafood package with a per-person price, bar add-on, and a follow-up if no reply within 48 hours.
