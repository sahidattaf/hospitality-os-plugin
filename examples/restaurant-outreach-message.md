# Example: Restaurant Outreach Message

## Input

BOSSA Asado i Mar wants to start a recurring weekly order with a local seafood supplier in exchange for a "local catch" menu credit and social tag.

## Expected output

- Target snapshot (supplier, what they likely value).
- Value proposition framed around reliability and exposure.
- Opening message referencing something specific about the supplier.
- Two-step follow-up sequence for no response.
- Suggested next step with timing.
