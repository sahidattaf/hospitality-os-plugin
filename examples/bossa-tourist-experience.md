# Example: BOSSA Tourist Experience Package

## Input

```text
Venue: BOSSA Asado i Mar
Audience: Cruise-ship visitors with limited time on shore
Highlights: sunset views, asado and seafood tasting
Need: itinerary slot of about 90 minutes
```

## Expected output

- Traveler snapshot (what cruise visitors are looking for).
- Experience package concept ("Sunset Asado Experience").
- Itinerary slot details: timing, menu highlights, add-ons.
- Guest-facing description.
- Logistics checklist (reservations, group sizes, transport coordination with tour operators).
