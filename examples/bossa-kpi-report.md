# Example: BOSSA KPI Report

## Input

```text
Venue: BOSSA Asado i Mar
Period: this week vs. last week
Average check: $42 (last week $38, target $40)
Food cost: 31% (target 28%)
Labor cost: 24% (target 24%)
Note: octopus price increased this week
```

## Expected output

- KPI table: metric, value, prior period, target, status.
- Key drivers (e.g., octopus price increase pushing food cost over target).
- Recommended actions for any KPI off track (e.g., review octopus portioning or pricing).
