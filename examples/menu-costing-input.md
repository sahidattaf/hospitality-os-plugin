# Example: Menu Costing Input

## Input

```text
Venue: BOSSA Asado i Mar
Item: Seafood skewer
Ingredients (per portion):
- Shrimp: 120g @ $0.02/g
- Octopus: 80g @ $0.025/g
- Peppers and onion: 50g @ $0.004/g
- Marinade: $0.30 per portion
Yield/waste factor: 10% trim loss on seafood
Target food cost: 28%
Current menu price: $18
```

## Expected output

- Recipe cost breakdown per ingredient.
- Cost per portion including waste factor.
- Food cost percentage at the current price.
- Comparison to the 28% target.
- Pricing or portion recommendation to hit target margin.
