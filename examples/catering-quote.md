# Example: Catering Quote

## Input

```text
Venue: BOSSA Asado i Mar
Inquiry: 50-guest birthday dinner on the beach
Date: flexible, planner has not confirmed
Notes: planner mentioned "family-style" and asked about bar options
```

## Expected output

- Qualification summary and missing-info questions (date, budget, dietary needs).
- Recommended package: family-style asado and seafood, with bar add-on.
- Itemized quote with per-person pricing and any minimums.
- Proposal draft.
- Follow-up plan if no response within 48 hours.
