# Example: BOSSA Fire Chef Menu Concept

## Input

```text
Venue: BOSSA Asado i Mar
Available: octopus, short rib, seasonal vegetables (peppers, onion, plantain)
Station: wood-fire grill
Goal: New shared platter, mid-to-high price band
```

## Expected output

- Dish concept ("Fuego Mixto" style shared platter).
- Kitchen spec: ingredients, portioning, marinade/rub, cook time/temperature, plating.
- Pairing suggestion (sauce, side, or beverage).
- Execution notes/risks and a fallback if an ingredient runs short.
- Guest-facing menu description.
