# Example: BOSSA Owner Dashboard

## Input

```text
Venue: BOSSA Asado i Mar
Period: past week
Covers: up 8% vs. prior week, driven by a busy Saturday
Top seller: seafood skewer
Issue: octopus inventory ran low on Friday
Feedback: one guest complaint about slow service
```

## Expected output

- Headline numbers with comparison to prior period.
- Highlights (busy Saturday, top-selling dish).
- Risks/flags (Friday octopus shortage, slow-service complaint).
- Decisions needed this week, with recommendations (e.g., adjust octopus par level, review Friday staffing).
- One-line overall read on the week.
