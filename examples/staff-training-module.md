# Example: Staff Training Module

## Input

A new server is starting their first solo shift at BOSSA Asado i Mar and needs a one-shift onboarding module covering menu knowledge, upsell language, and a readiness check.

## Expected output

- Training objective.
- Module breakdown (menu story, asado and seafood pairings, upsell language for the dessert trio).
- Step-by-step content per module.
- A 5-question knowledge check / checklist.
- Follow-up coaching points and re-check timeline.
