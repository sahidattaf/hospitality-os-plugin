# Example: BOSSA Fire Box Offer

## Input

```text
Venue: BOSSA Asado i Mar
Occasion: Date night, 2 guests
Items under consideration: ribeye, seafood skewers, chimichurri, side
Channel: pickup and delivery (20% platform fee on delivery)
```

## Expected output

- Bundle contents for "Fire Box for Two".
- Cost and pricing breakdown, including margin after the 20% delivery fee.
- Packaging notes (what travels well, what's packed separately).
- Reheating/finishing instructions for the guest.
- Guest-facing description of the box.
