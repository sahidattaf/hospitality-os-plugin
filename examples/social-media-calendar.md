# Example: Social Media Content Calendar

## Input

BOSSA Asado i Mar wants a one-week content calendar promoting a new sunset dinner menu, including a new seafood skewer, a sunset dinner reel, a guest review repost, and a weekend reservation reminder.

## Expected output

- Content calendar table (date, channel, concept).
- Draft captions in BOSSA's warm, coastal voice.
- Shot list / visual notes for posts needing new media.
- Hashtag or tagging suggestions.
